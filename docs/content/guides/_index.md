Each guide is built around a job rather than a command. They assume you have run
the [quick start](/getting-started/quick-start/).

- [Resource URIs](/guides/resource-uris/) walks the record graph and shows how to
  use booking as a `database/sql`-style driver so a host can address Booking.com
  as `booking://` URIs.
- [Add a command](/guides/adding-a-command/) is for contributors: model a record
  and turn it into a command, a route, and a tool at once.
